## 🚀 启动步骤

### 1. 启动后端服务
```bash
cd backend
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```
**访问**: http://localhost:8000/docs (Swagger UI)

### 2. 启动前端服务
```bash
cd frontend-demo
npm run dev
```
**访问**: http://localhost:5173

### 3. 验证服务健康
```bash
curl http://localhost:8000/health
```
**预期响应**: `{"status":"healthy","services":{"database":"connected","dashvector":"connected","redis":"connected"}}`

### 4. （可选）启动 Docker 环境
```bash
docker-compose up -d
```
**访问**:
- 前端: http://localhost:5173
- 后端: http://localhost:8000/docs
