# 对话式提示词优化（2026-03-16）

## 任务背景
用户反馈聊天提示语句不够口语化，且聊天未必都是提问，需要从已有音频中提取可以形成上下文聊天的语句作为提词库。

## 完成的工作

### 1. 创建自然对话短语库
- **文件**: `backend/services/natural_conversational_phrases.py`
- **内容**: 包含150+个自然的口语化语句，分为：
  - 日常问候和开场
  - 广播节目常见开场
  - 交通相关（《一路畅通》风格）
  - 生活话题
  - 新闻时事
  - 情感交流
  - 经验分享
  - 陈述感受
  - 广播节目特色语句
  - 更口语化的表达
  - 互动邀请
  - 从真实音频中提取的示例短语（基于广播内容）

### 2. 改进对话式提示词生成服务
- **文件**: `backend/services/conversational_prompt_service.py`

#### 关键改进：
1. **增强自然短语提取** (`extract_natural_phrases`函数):
   - 改进句子分割逻辑，支持更多中文标点
   - 添加`is_conversational_sentence`函数判断口语化句子
   - 支持从长句中提取自然子短语
   - 返回更多自然短语（最多8个）

2. **优化对话语句生成** (`generate_conversational_statement`函数):
   - **减少提问比例**: 从约50%降至10%
   - **增加陈述句比例**: 提高到40%
   - **添加加权随机选择**: 控制不同类型语句的概率
   - **直接使用自然短语库**: 30%概率直接从短语库选取
   - **更口语化的模板**: 添加"哎"、"说实话"、"其实吧"等口语化开头

3. **更新默认建议列表** (`get_default_conversational_suggestions`函数):
   - 使用自然短语库作为主要来源
   - 补充精选的口语化语句
   - 随机打乱顺序，返回80条丰富建议

### 3. 更新微信小程序默认提示语句
- **文件**: `wechat-miniprogram/pages/chat/chat.ts`
- **修改**: 替换硬编码的`quickQuestions`列表为更口语化的语句
- **新列表特点**: 减少提问，增加陈述句，使用更自然的表达方式

### 4. 确保后端服务使用改进的逻辑
- `backend/services/chat_service.py`中的`generate_chat_suggestions`函数已经调用改进后的服务
- API端点`/api/v1/chat/suggestions`返回更自然的口语化建议

## 核心改进效果

### 1. 口语化程度提升
- 使用更多日常口语表达："哎，我跟你说"、"说实话"、"其实吧"
- 减少正式提问，增加自然陈述
- 包含广播节目特色语句，更贴近音频内容

### 2. 语句类型多样化
- **陈述句**: 40% (例如："今天天气不错"、"我觉得挺有意思的")
- **感叹句**: 20% (例如："哇，这个话题不错！")
- **评论句**: 20% (例如："我对这个有些看法")
- **自然疑问句**: 10% (例如："你对这个有什么了解吗？")
- **分享句**: 10% (例如："我最近听说一些消息")

### 3. 上下文聊天能力
- 提供可以形成对话的语句，不仅仅是独立提问
- 包含回应性语句："我也有同感"、"说得有道理"
- 包含话题延续语句："说到这个，我想起来一件事"

### 4. 基于音频内容的关联性
- 包含从广播节目内容提取的示例短语
- 交通话题（《一路畅通》风格）："北京今天早高峰特别堵"
- 旅游话题（《行走天下》风格）："这个地方风景特别美"
- 广播节目特色："接下来是广告时间"、"欢迎回到节目"

## 技术实现细节

### 加权随机选择算法
```python
template_weights = {
    "statement": 0.4,      # 40%
    "exclamation": 0.2,    # 20%
    "comment": 0.2,        # 20%
    "natural_question": 0.1, # 10%
    "sharing": 0.1,        # 10%
}
```

### 口语化句子判断逻辑
- 检查常见口语化开头模式（正则表达式）
- 检查话题词汇
- 检查句子结构和长度
- 支持中文口语结尾词："吧"、"嘛"、"呢"等

### 短语提取优化
- 支持从长句中按逗号分割提取子短语
- 过滤太短或太长的短语
- 去重处理，避免重复建议
